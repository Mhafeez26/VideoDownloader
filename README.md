# Video Downloader

A simple and efficient Video Downloader application that allows users to download videos from online platforms by providing a URL.

---

## Features
- Download videos using URL
- Supports multiple formats (MP4, etc.)
- Fast and lightweight processing
- Simple user interface (HTML/Python based)
- Easy to use

---

##  Tech Stack
- Python (Flask / backend)
- HTML, CSS, JavaScript (frontend)
- yt-dlp / video processing library
